# Project Vision

Diag is a professional-grade Windows utility for collecting
hardware identifiers and diagnostic information in a structured, exportable form.

The tool prioritizes:
- Accuracy over convenience
- Defensive data collection
- Clear reporting of unavailable or restricted fields

## Non-goals
- Background services
- Automatic cloud upload (future opt-in only)
